# Task29
Напиши что такое регулярные выражения (Regular Expressions) и для чего используются в Java
https://docs.oracle.com/javase/tutorial/essential/regex/
Реализуй метод для проверку имени пользователя на валидность с использованием регулярного выражения.
Количество символов в имени пользователя не должно быть меньше четырех и больше шестнадцати. В имени  должны использоваться только буквы латинского алфавита, цифры и символ нижнего подчеркивания. В случае не прохождения проверки в консоль должно выводиться соответствующее уведомление.
 
Также реализуй еще один метод для проверки имени пользователя на валидность но без использования регулярного выражения.
Проанализируй и опиши преимущества и недостатки использования регулярного выражения.
